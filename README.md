# 简介
教务系统

# 使用技术
IOC容器：Spring
Web框架：SpringMVC
ORM框架：Mybatis
安全框架：Shiro
数据源：C3P0
日志：log4j
前端框架：Bootstrap

# 快速上手
### 1、运行环境和所需工具
* 编译器：IntelliJ IDEA
* 项目构建工具：Maven
* 数据库：Mysql
* JDK版本：jdk1.8
* Tomcat版本：Tomcat8.x
### 2、初始化项目
* 在你的Mysql中，创建一个数据库名称为 examination_system 的数据库，并导入sql 文件,
* 进入src/main/resources修改mysql.properties配置文件,把数据库登录名和密码，改为你本地的
* 使用 IntelliJ IDEA 或 Eclipses导入项目
* 运行
